CUCM v14 / v15 — Smart Licensing

CUCM v14+ uses Cisco Smart Software Licensing only (no PLM/ELM). User-based UCL via CSSM with tier substitution hierarchy. CSSM hierarchy: CUWL Professional ⊇ CUWL Standard ⊇ Enhanced Plus ⊇ Enhanced ⊇ Basic ⊇ Essential.

Device → License rules: 1 device = Enhanced minimum · 2 devices = Enhanced Plus minimum · 3–10 devices = CUWL minimum · Room device = TelePresence Room · No OwnerUserID = Device Only license
UCL — User Connect Licensing (Smart License)
Cisco 3905, 6901, ATA 191/192/187. Jabber IM/Presence. No SNR, no video.
Cisco 6921. SNR/Mobile Connect. Jabber. No video.
79xx, 89xx, 99xx, 6941/6961, DX80, TelePresence EX60/EX90. Jabber Desktop+Mobile. SNR. 1 device max.
All Enhanced features + 2nd device (e.g. desk phone + Jabber softclient). Minimum for dual-device users.
CUWL — Cisco Unified Workspace Licensing
3+ devices per user. Jabber Desktop+Mobile. SNR. IM&P. Advanced UC features. Does NOT include Webex Meetings or Unity Connection.
All CUWL Standard + Webex Meetings + Unity Connection VM. Max UC suite. 10 devices per user.
Cisco TelePresence 3200/3000/1300, MX Series, TX Series, SX Series. Room-based (not user-based). Also covers Webex Board/Room/Desk registered to CUCM.
Devices with blank OwnerUserID field. License determined by device type (usually Essential or Basic).

SME & Unity Connection

Session Management Edition (SME)
SME centralises the dial plan across multiple CUCM leaf clusters. One SIP trunk from SME to LGW replaces N per-cluster trunks — significantly reducing CUBE session license cost.
Infrastructure license — CCW-quoted. Enter monthly equivalent. SME requires Smart Licensing.
SME saves 0 CUBE session licenses vs per-cluster trunking (one LGW trunk instead of N).
Unity Connection (CUC) Voicemail
CUC voicemail is INCLUDED in CUWL Professional licenses. Only add separate CUC licenses for UCL users who need voicemail.
Basic voicemail + VM-to-email. For UCL Essential/Basic/Enhanced users only.
Adds Exchange/O365 calendar, Outlook voice access, visual voicemail to standard VM.
Non-personal shared voicemail boxes (AA, HG, dept). No user login capability.
