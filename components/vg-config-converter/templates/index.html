Upload or paste an existing VG350 running-config. The tool remaps legacy slot/subslot/port references into flat 0/0/N numbering — and generates a sortable Excel workbook with OLD_Config and NEW_Config sheets mapping each port to its extension.

Port mapping logic

Legacy format: slot/subslot/port
e.g. 0/2/13 with base slot=0, 24 ports/module

New format: 0/0/N
N = (module_slot − base_slot) × ports_per_module + port
→ 0/0/61
